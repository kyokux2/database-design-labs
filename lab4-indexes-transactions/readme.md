# Лабораторная работа №4
## Индексы, EXPLAIN и транзакции

В работе использована база данных fitness_lab
(ЛР №1–№3).

## 1. Индексы
Созданы индексы для:
- поиска по диапазону дат
- фильтрации и сортировки
- поиска по подстроке (LIKE)

Для каждого индекса выполнен анализ
EXPLAIN ANALYZE до и после создания индекса.

## 2. Анализ производительности
Подготовлены запросы средней и высокой сложности
с JOIN, фильтрацией и сортировкой.
Показано изменение планов выполнения
(Seq Scan → Index Scan).

## 3. Транзакции
Рассмотрены аномалии:
- Dirty Read (невозможна в PostgreSQL)
- Non-repeatable Read
- Phantom Read
